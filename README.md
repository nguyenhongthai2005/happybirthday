# 🎂 Birthday Page - Ngọc Ánh

Trang chúc mừng sinh nhật được deploy bằng GitHub Pages.

## Deploy lên GitHub Pages

1. Tạo repo mới trên GitHub (ví dụ: `birthday-anh`)
2. Upload file `index.html` lên repo
3. Vào **Settings → Pages → Branch: main → Save**
4. Link sẽ là: `https://username.github.io/birthday-anh`
